Data Transfer Utility Help

Upload Screen

Feedback
This link appears on several pages throughout the transfer utility. Selecting it at anytime takes you to a screen where you can provide feedback about the site. We are constantly striving to improve this tool so your comments and suggestions are welcomed and encouraged.

Close Window

Change Password
You will only see this link if your user permissions allow for it. You can click this link to go to a page where you may change your password.

Close Window

First Name
Enter the first name of your client or the person for whom you want to upload a file. This will be used in the notification e-mail sent to your client.

Close Window

Last Name
Enter the last name of your client or the person for whom you want to upload a file. This will be used in the notification e-mail sent to your client.

Close Window

Email
It is critical this information be correct as this is the e-mail address to which a notification e-mail will be sent. Enter this information carefully. If your recipient is in your contacts list you may even want to copy and paste that information into this field.

Close Window

File Description
The information you enter here will be used in the notification e-mail to tell your client what file you have uploaded. Enter enough information so your recipient understands what you are sending.

Close Window

Upload File Field
This is the physical path to the file you want to upload into the transfer system. Although you can type the path yourself it is highly recommended you use the "Browse" button. If the path is not correct an error will occur and the file will not be uploaded.

Close Window

Browse
Click this button to browse for the file you want to upload. This feature uses the file browser of your operating system which insures the file path is correct.

Close Window

Upload Button
Selecting this button will begin the file upload.

Close Window

Logout
To immediately logout of the system click this button. You will be returned to the login page.

Close Window

Login Screen

Login User Name
Your user name is dependent upon whether you are a client of NRC or an NRC Associate. NRC clients should enter the user name they were provided in their notification e-mail. NRC Associates should enter the user name they were assigned.

Close Window

Login Password
Like the user name NRC clients should enter the password they were given in their notification e-mail. NRC Associates should enter the password they chose or were assigned.

Close Window

Sign In Button
After entering your user name and password click this button to enter the system. If your credentials are not valid you will be returned to this screen to re-enter your information.

Close Window
