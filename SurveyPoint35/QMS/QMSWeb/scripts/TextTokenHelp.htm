Script Screen Text Token Help

Text Token Description
RESPONDENT(FULLNAME) Returns respondent first and last name
RESPONDENT(PRONOUN_SUBJECT) Returns "he" or "she" depending on gender of respondent
RESPONDENT(PRONOUN_OBJECT) Returns "him" or "her" depending on gender of respondent
RESPONDENT(PRONOUN_POSSESIVE) Returns "his" or "her" depending on gender of respondent
RESPONDENT(PRONOUN_POSSESIVE2) Returns "his" or "hers" depending on gender of respondent
RESPONDENT(PRONOUN_REFLEXIVE) Returns "himself" or "herself" depending on gender of respondent
RESPONDENT(TITLE) Returns "Mr." or "Ms." depending on gender of respondent
RESPONDENT(RespondentID) Returns the respondent's id number
RESPONDENT(SurveyInstanceID) Returns the respondent's survey instance id number
RESPONDENT(FirstName) Returns the respondent's first name
RESPONDENT(MiddleInitial) Returns the respondent's middle initial
RESPONDENT(LastName) Returns the respondent's last name
RESPONDENT(Address1) Returns the respondent's first address line
RESPONDENT(Address2) Returns the respondent's second address line
RESPONDENT(City) Returns the respondent's city
RESPONDENT(State) Returns the respondent's state
RESPONDENT(PostalCode) Returns the respondent's zip code
RESPONDENT(TelephoneEvening) Returns the respondent's evening telephone number
RESPONDENT(TelephoneDay) Returns the respondent's day telephone number
RESPONDENT(Email) Returns the respondent's email address
RESPONDENT(DOB) Returns the respondent's date of birth
RESPONDENT(Gender) Returns the respondent's gender
RESPONDENT(ClientRespondentID) Returns the respondent's client respondent id number
RESPONDENT(SSN) Returns the respondent's social security number
RESPONDENT(BatchID) Returns the respondent's batch id number
RESPONDENT(SurveyInstanceName) Returns the respondent's survey instance name
RESPONDENT(ClientName) Returns the respondent's client name
RESPONDENT(SurveyName) Returns the respondent's survey name
RESPONDENT(ClientID) Returns the respondent's client id number
RESPONDENT(SurveyID) Returns the respondent's survey id number
RESPONDENT(CallsMade) Returns the number of telephone calls made to the respondent
RESPONDENT(Final) Returns "1" if the respondent is final or "0" if the respondent is not final
RESPONDENT(NextContact) Returns the respondent's earliest contact date and time allowed by the system
PROPERTY(property name) Returns the respondent's specified property value
ANSWERTEXT(short question name) Returns the respondent's answer text for the specified question
RESPONSETEXT(short question name) Returns the respondent's response text for the specified question
USERNAME() Returns the operator's name
OTHER(variable name) Returns the specified variable value
QUERY(sql statment) Returns the results of the sql statement. Use {0} for respondent id and {1} for script id within sql statement
