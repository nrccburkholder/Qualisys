Trigger Code Help

Command Description
%EXIT Exits interview without running completeness calculation and end script trigger
%END End interview and runs completeness calculation and end script trigger
%NEXT Go to next screen. Use in pre-screen trigger to skip the current screen
%SCREENINDEX(screen index) Go to specified screen index
%SCREENID(script screen id) Go to specified script screen id
sql statement Executes sql statement
{0} Used in sql statment for respondent id number
{1} Used in sql statment for script id number
{2} USed in sql statement for user id number

Criteria Help

Criteria Type Description
Simple Simple match to answer category.
Text Criteria text must match exactly with open answer text.
TextLike Criteria text can be anywhere in open answer text.
Parameter Criteria text must match exactly the parameter value (see Valid Criteria Parameters list below).
ParameterLike Criteria text can be anywhere in the parameter value (see Valid Criteria Parameters list below).
Parameter > Criteria text must be greater than the parameter value (see Valid Criteria Parameters list below).
Parameter < Criteria text must be less than the parameter value (see Valid Criteria Parameters list below).
And All criteria within AND must be true
Or At least one criteria within OR must be true
Unanswered Question is unanswered or has no response
